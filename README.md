# Nuvio Unofficial — AltStore source

An auto-updating AltStore / SideStore / Feather source for the
**Full** and **Enhanced** Nuvio iOS builds from
[`luqmanfadlli/NuvioMobile-iOS`](https://github.com/luqmanfadlli/NuvioMobile-iOS).

## Add to AltStore

Add this source URL in your app → **+**:

```
[https://raw.githubusercontent.com/<your-user>/<this-repo>/main/apps.json](https://raw.githubusercontent.com/FlashLightning67/Nuvio-Altstore-Repo/refs/heads/main/apps.json)
```
